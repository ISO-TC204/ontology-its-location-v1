# ITS Ontology for Location Referencing

The ITS Ontology for Location Referencing consists of the following patterns:

- [Area](classes/AreaPattern.md)
- [Core](classes/LocationCorePattern.md)
- [Itinerary](classes/ItineraryPattern.md)
- [Linear](classes/LinearPattern.md)
- [Point](classes/PointPattern.md)

The formal definition of this ontology is available in [TURTLE Syntax](its-location.ttl).
